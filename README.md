# laravel-react-todo
An example TODO app made with Laravel and React.


## Installation steps
1. Installing assets
```
$ composer install
$ npm install
```
2. Environment configuration based on .env.example template file
3. Database setup
```
$ php artisan migrate
$ php artisan db:seed
$ php artisan key:generate
$ php artisan serve
```
